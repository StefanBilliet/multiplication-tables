## User Story 5 - Unlock new tables progressively

As a child,  
I want to unlock new multiplication tables over time,  
so that I feel like I am making progress.

### Acceptance Criteria

- Given I complete practice successfully for the current table,
  when I meet the unlock rule,
  then the next table becomes available.

- Given a new table has been unlocked,
  when I return to the start screen,
  then that table is shown as available.

- Given a table has not yet been unlocked,
  when I return to the start screen,
  then it remains unavailable.
