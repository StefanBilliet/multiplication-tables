## User Story 2 - Answer multiplication questions

As a child,  
I want to answer multiplication questions one by one,  
so that I can practice the selected table.

### Acceptance Criteria

- Given I have started a practice session,
  when a question is shown,
  then I see exactly one multiplication question at a time.

- Given I am practicing a specific table,
  when questions are shown,
  then every question belongs to that selected table.

- Given I submit an answer,
  when the answer is accepted,
  then the app evaluates whether it is correct or incorrect.
