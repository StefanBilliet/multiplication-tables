## User Story 3 - Receive immediate feedback

As a child,  
I want to immediately see whether my answer is correct,  
so that I know how I am doing.

### Acceptance Criteria

- Given I submit a correct answer,
  when the app evaluates it,
  then I see clear positive feedback.

- Given I submit an incorrect answer,
  when the app evaluates it,
  then I see clear incorrect feedback.

- Given I submit an incorrect answer,
  when feedback is shown,
  then the correct answer is also shown.

- Given feedback has been shown,
  when it finishes,
  then the app proceeds to the next question unless the session is complete.
