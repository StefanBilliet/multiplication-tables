## User Story 3 - Complete a short practice session

As a child,  
I want practice to happen in short sessions,  
so that it feels manageable and fun.

### Acceptance Criteria

- Given I start a practice session,
  when I answer questions,
  then the session ends after a fixed number of questions.

- Given the final question of the session has been answered,
  when the session ends,
  then I see a summary screen.

- Given the summary screen is shown,
  when I view it,
  then I can see how many answers were correct.
