## User Story 1 - Start a practice session for a chosen table

Status: Finished

As a child,  
I want to choose a multiplication table to practice,  
so that I can focus on learning one table at a time.

### Acceptance Criteria

- Given I am on the start screen,
  when I select an available multiplication table,
  then the app starts a practice session for that table.

- Given a table is still locked,
  when I view the start screen,
  then I can see that the table is not yet available.

- Given I have unlocked a table,
  when I return to the start screen,
  then I can start a session for that table again.
