# Multiplication Tables

Scaffold for a Vite 8 + React + TypeScript progressive web app.

## Included

- Mantine
- Vitest
- React Testing Library
- Storybook
- Fishery
- Biome
- PWA basics with a manifest and service worker

## Scripts

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run test`
- `npm run test:watch`
- `npm run storybook`
- `npm run build-storybook`
- `npm run check`
- `npm run format`
